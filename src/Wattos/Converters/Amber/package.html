General
The only source files here that were created by humans are the java sourcefiles: StarOut.java and Utils.java and the JavaCC grammar file AmberParser.jj.All other files are (re-)generated using JavaCC Parser Generator.

To-do:

  • Ignored r1, r2, and ir6 values for distances.
  • Ignored the tags grnam1(1) etc. because they don't occur (as of 11/4/2004) in MR files.
  • Some values seem to have to be remembered going from 1 restraint to the next. E.g. entry 1dgq only has a force constant (20) defined within the first restraint.
  • Ignored tags r1 and r4 which are switching distances for dihedral angle restraints important for the shape of the restraint potential.
  • For RDCs the dij tag (for the internuclear vector size) is ignored.
