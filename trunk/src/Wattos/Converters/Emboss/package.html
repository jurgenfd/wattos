Parser for EMBOSS formatted restraint files.

The only source files here that were created by humans are the java source files: StarOut.java and Utils.java and the JavaCC grammar file EmbossParser.jj. All other files are (re-)generated using JavaCC Parser Generator.

Examples of input are for distances:

;HA(I)-HN(I+1)

# PDB entry: 1bhi

# PDB entry: 1d8j
       1   7 GLY  HN   1   6 SER  HA    1.0  1.0  1.90  4.00
