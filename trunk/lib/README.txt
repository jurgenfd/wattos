This directory contains jar files used by different parts of the Wattos package
and the Wattos.jar itself.

Content files           Description
------------------------+-------------------------------------------------------
AbsoluteLayout.jar		Swing related small jar.
CSVutils.jar			CSV files
JFlex.jar				Parser generation
JavaCC.jar				Parser generation
Jmol.jar				Molecular imagery (not essential for anything)
README.txt				This file.
Wattos.jar				Contains all classes in project.
Wattos.jardesc			Used by Eclipse to generate the Wattos.jar (to be obsoleted?)
ant-contrib.jar			Special ant tasks
colt.jar				Primitive datatypes for arrays and hashes.
gnujaxp.jar				No idea.
itext-1.4.jar			Required by jfreechart
jakarta-regexp.jar		Regular expressions from before Java 1.4.
jcommon-1.0.0.jar		Required by jfreechart
jdbc20x.jar				Java DB connectivity
jfreechart-1.0.1.jar	Plotting tool
mysql-connector-java-3.0.16-ga-bin.jar	Mysql connector (obsolete?)
mysql-connector-java-5.0.3-bin.jar		Mysql connector (use this one when in doubt)
printf_hb15.jar			Formatted printing like c.
servlet-api.jar			Servlet Api
starlibj_with_source.jarSteve's lib to STAR
swing-layout-1.0.jar	Swing related specific for Sjors Webapp.

NB The binaries here should be CVS checked in as binaries otherwise there might be problems.
